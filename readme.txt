为WordPress提供快速的开发模块、工具包，减少主题开发工作量。

== Description ==

主题开发总是会做一些重复的工作，WP SHARE MODULES试图通过一个插件，解决掉一些经常会遇到的开发问题，通过一些小模块，实现快速的工具包，比如去除wp_head输出的多余信息，只需要后台一键就可以解决，同时为主题开发提供扩展函数，比如the_post_time, get_excerpt等，这些函数都是我们经常想要的。

开发与探讨：http://github.com/tangshuang/wp-share-modules
如果你在使用中遇到一些问题，可以在这个github页提交issue。

== Installation ==

1、在WordPress后台“安装插件”搜索“WP SHARE MODULES”；或者，下载插件解压后上传到/wp-content/plugins/目录<br />
2、在后台插件列表中启用它<br />
3、在“插件-SHARE MODULES”菜单中，启用你想要的功能

<strong>注意：如果启用某个模块后整个博客打不开，那么请在wp-config.php中启用DEBUG模式，获得错误信息，并在上面的github中提交issue。</strong>
